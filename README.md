# ChatGPT News Today

A clean bilingual (Chinese and English) Hugo site powered by Blowfish, focused on ChatGPT information and news.

## Stack

- Hugo + Blowfish (Hugo Module)
- GitHub Pages (GitHub Actions)

## Local development

1. Install Hugo Extended.
2. Run:

```bash
hugo server
```

## Deployment

Push to `main`, then GitHub Actions deploys the site to Pages automatically.
