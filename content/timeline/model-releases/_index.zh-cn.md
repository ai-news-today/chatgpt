## 2026

| **时间**       | **模型/产品**                   | **核心功能与突破（来源：OpenAI API Changelog）**                                                                                                                                                                                       |
| -------------- | ------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **2026-02-24** | **gpt-5.3-codex**               | 上线 `gpt-5.3-codex`（Responses API）。                                                                                                                                                                                                |
| **2026-03-03** | **gpt-5.3-chat-latest**         | `gpt-5.3-chat-latest` 上线（Chat Completions / Responses API），指向 ChatGPT 当前使用的最新 GPT‑5.3（Instant）快照。                                                                                                                   |
| **2026-03-05** | **gpt-5.4 / gpt-5.4-pro**       | 上线 `gpt-5.4`（Chat Completions / Responses）与 `gpt-5.4-pro`（Responses）；并带来 Tool search、内置 Computer use、\(1M\) 上下文窗口与原生 Compaction 等长对话/代理工作流能力。                                                       |
| **2026-03-17** | **gpt-5.4-mini / gpt-5.4-nano** | 上线 `gpt-5.4-mini` 与 `gpt-5.4-nano`（Chat Completions / Responses）：mini 面向高并发更高性价比场景并支持 tool search / computer use / compaction；nano 面向更简单的高并发任务，支持 compaction 但不支持 tool search / computer use。 |

| **时间**            | **模型/产品**                   | **核心功能与突破（来源：ChatGPT 发布说明）**                                                                                            |
| ------------------- | ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **2026-03（逐步）** | **GPT-5.4 Thinking（ChatGPT）** | ChatGPT 端推出 GPT‑5.4 Thinking（以“Thinking”口径呈现）；用于推理/编程/代理式任务。注：ChatGPT 的命名/节奏与 API 机型名可能不完全一致。 |
| **2026-03（逐步）** | **GPT-5.4 mini（ChatGPT）**     | ChatGPT 端逐步推出；在部分方案/场景下作为 GPT‑5.4 Thinking 触及速率限制时的回退模型。                                                   |

## 2025

| **时间**       | **模型/产品**              | **核心功能与突破**                                                                       |
| -------------- | -------------------------- | ---------------------------------------------------------------------------------------- |
| **2025-01-31** | **o3-mini**                | 高性价比推理模型，针对编程/数学/科学优化，并支持结构化输出与函数调用等开发者能力。       |
| **2025-02-27** | **GPT-4.5（研究预览）**    | 更自然的对话体验与更强的用户意图遵循，并降低幻觉。                                       |
| **2025-04-16** | **OpenAI o3 / o4-mini**    | 推理模型更新：o3 强推理，o4-mini 更高性价比与更高使用上限。                              |
| **2025-05-14** | **GPT-4.1 / GPT-4.1 mini** | 编程导向模型进入 ChatGPT；4.1 mini 替代 4o mini。                                        |
| **2025-06-10** | **OpenAI o3-pro**          | 更长时间思考以提升可靠性；面向 Pro 用户开放，也可在 API 使用。                           |
| **2025-08-07** | **GPT-5**                  | 新一代旗舰模型逐步推出；付费用户可选 GPT-5 或 GPT-5 Thinking，Pro/Team 有 Thinking Pro。 |
| **2025-11-12** | **GPT-5.1**                | GPT-5 更新为 5.1 Instant/Thinking，并提供更强风格与个性化控制。                          |
| **2025-12-11** | **GPT-5.2**                | 推出 GPT-5.2（Instant/Thinking/Pro），并更新知识截止日期（官方说明为 2025 年 8 月）。    |
