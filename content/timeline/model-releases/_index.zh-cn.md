## 2026

| **时间**    | **模型/产品**           | **核心功能与突破**                                                    |
| ----------- | ----------------------- | --------------------------------------------------------------------- |
| **2026.01** | **GPT-5.2-Codex**       | **自主编码代理**，能够理解整个代码库并自主执行多步工程任务。          |
| **2026.02** | **GPT-5.3-Codex-Spark** | 实时生成模型，支持极速编码反馈，延迟降至毫秒级。                      |
| **2026.03** | **GPT-5.4 / 5.4-mini**  | **引入“电脑使用”功能**，模型可直接在用户电脑上进行跨 App 自动化操作。 |
| **2026.03** | **Sora 2.0 (API 升级)** | 支持 1080p 输出、固定角色一致性及长达 20 秒的连续视频生成。           |

## 2025

| **时间**    | **模型/产品**               | **核心功能与突破**                                                    |
| ----------- | --------------------------- | --------------------------------------------------------------------- |
| **2025.01** | **o3-mini**                 | 高性价比推理模型，在 STEM 任务上性能超过 o1-mini 且速度更快。         |
| **2025.02** | **GPT-4.5 (Orion)**         | 侧重“情商”与“审美”的预训练大模型，具备更强的创意写作与直觉判断力。    |
| **2025.08** | **GPT-5.0 (旗舰版)**        | **全能旗舰**，统一了 o 系列的推理与 4o 的速度，支持更长的上下文记忆。 |
| **2025.12** | **GPT-5.2 / GPT-Image-1.5** | 推出新一代图像引擎，支持更精准的局部编辑与 4 倍生成速度。             |

## 2024

| **时间**    | **模型/产品**            | **核心功能与突破**                                                |
| ----------- | ------------------------ | ----------------------------------------------------------------- |
| **2024.02** | **Sora (首次亮相)**      | 文本转视频模型，支持生成 60 秒高清视频，模拟真实物理世界。        |
| **2024.05** | **GPT-4o (Omni)**        | **原生多模态**，实现文本、音频、图像实时无缝交互，延迟极低。      |
| **2024.09** | **o1-preview / o1-mini** | **推理模型开端**，通过强化学习和思维链处理复杂的数学、编程任务。  |
| **2024.12** | **ChatGPT Search**       | SearchGPT 正式集成进 ChatGPT，实时联网并提供引用源，挑战 Google。 |

## 2023

| **时间**    | **模型/产品**         | **核心功能与突破**                                       |
| ----------- | --------------------- | -------------------------------------------------------- |

| **2023.03** | **GPT-4**             | 推理能力跨越式提升，首次支持图像理解（多模态初步尝试）。 |
